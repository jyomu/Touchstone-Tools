Touchstone Tools

Touchstone ファイルを扱うブラウザツール集

TOOL

Touchstone Converter

Touchstone ファイルのフォーマット変換・ポート並べ替えなどを行います。

 TOOL

Touchstone CSV Extractor

Touchstone ファイルから Sパラメータを CSV 形式で書き出します。

 TOOL

Touchstone Waveform Analyzer

Sパラメータを波形グラフ・スミスチャートで可視化。マーカー・ズーム・群遅延・帯域幅検出など実務向け解析機能を搭載。

 TOOL

Touchstone Mixed-Mode Converter

シングルエンド Sパラメータを Mixed-mode（差動 Sdd / コモン Scc / 変換 Scd）に変換します。偶数ポート（2〜8）対応。
All processing runs locally in your browser. No data is sent to any server.
